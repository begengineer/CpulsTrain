# C++の勉強記録

## 2026/03/30
以下の勉強を実施
 - カプセル化
 - コンストラクタ
 - 継承
 - 多態性
 - 演算子のオーバーロード(さわりのみ)
 - STL

## 2026/04/19
以下の勉強を実施
 - マルチスレッドプログラミング（`<thread>`, `<mutex>`）
   - プロセスとスレッドの仕組み
   - スレッドの生成・同期
 - ビット演算・メモリ操作（`memset`, `memcpy` の自作）
 - 非同期処理（`<future>`, `std::async`）
   - `std::future` / `std::promise` による結果の受け渡し
   - `std::launch::async` による並列実行
